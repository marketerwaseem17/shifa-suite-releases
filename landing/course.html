📦 Lesson 1

Installation & First Launch

Software install karna aur pehli dafa kholna — sirf 3 minute lagenge.

Requirements

💻 Operating SystemWindows 7, 8, 10, 11 (64-bit)
🖥️ RAMMinimum 2 GB (4 GB recommended)
💾 Storage500 MB free space
🌐 InternetSirf download ke liye — baad mein zarurat nahi

Installation Steps

1
Setup file download karein

Google Drive link ya apna diya gaya link khol kar Shifa-Suite-Setup-1.0.0.exe download karein.

2
Setup file chalayein

Downloaded file pe double click karein. Agar Windows "Unknown publisher" warning dikhaaye to Run Anyway click karein.

3
Install folder choose karein

Default folder theek hai (C:\Program Files\Shifa Suite). Next → Install click karein.

4
Installation complete

"Launch Shifa Suite" checked rakhein aur Finish click karein. App khul jayegi.

⚠️
Windows Defender: Kabhi kabhi antivirus new software ko block karta hai. Agar aisa ho to Start Menu → Windows Security → Virus & threat protection → aur Shifa Suite ko allow karein. Ya directly WhatsApp karein: 03109888092

First Launch

App pehli dafa kholein to aap dekhein gay:

  • 🎁 7-day free trial automatically shuru ho jata hai — koi key nahi chahiye
  • Sidebar mein "Free trial — X days left" dikhega
  • Setup wizard shuru hoga jahan aap clinic ki details daal sakte hain
✅
Desktop shortcut: Installation ke baad desktop pe "Shifa Suite" shortcut bana jata hai. Double click karein to open karein.

💾 Lesson 11

Data Backup & Restore

Aapka data sirf aapke computer mein hai — iska backup lena bohot zaruri hai.

⚠️
Important: Agar computer kharab ho gaya aur backup nahi liya to sara clinic data lose ho sakta hai. Hafte mein ek dafa backup lena acha practice hai.

Backup Kaise Lein

1
Settings kholein

Sidebar ke neeche Settings icon pe click karein.

2
"Data & Backup" tab select karein
3
"Create Backup" button dabayein

Ek file chooser khulega. Jahan save karna chahein wahan save karein.

4
File USB ya Google Drive mein copy karein

Backup file ko ek aur safe jagah copy karein — USB drive ya Google Drive.

Backup File Kahan Save Karein?

  • 🔌 USB Drive — most reliable, offline copy
  • ☁️ Google Drive — automatic cloud backup
  • 📧 Email to yourself — attachment ke tor par

Restore Karna (Data Wapas Laana)

1
Settings → Data & Backup kholein
2
"Restore Backup" dabayein

Apni backup file select karein.

3
Confirm karein

Warning padh kar OK dabayein — current data replace ho jayega. App restart hogi.

✅
Best Practice: Har Jumme (Friday) ko backup lena ek achi routine hai. Backup file ka naam date se rakhein jaise shifa-backup-2025-01-10.db
